### Develop

**We highly recommend using `pnpm` as the package manager during developing.**

#### Install Dependencies

```bash
pnpm install
```

#### Debug

```bash
pnpm dev
```

#### Format

```bash
pnpm format
```

#### Build

```bash
# This will create a dir named "dist" in your workspace folder.
# You can change the build options in tsconfig.
pnpm build
```
